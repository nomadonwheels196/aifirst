# GTM SOPs

## Purpose
Department‑specific SOPs for Go‑To‑Market (Marketing, Sales, Customer Success).

## How to add a new SOP
Copy `Company_SOPs/_TEMPLATE` into this `SOPs/` directory, rename, and fill in the spec and runbook. Link any material changes to `business_spec/`.

## Ownership & Review
GTM leadership owns this area; review at least quarterly.
