# Onboarding

## Purpose
Ensure new hires reach productivity quickly and consistently.

## Checklist
- Accounts: SSO/MFA, email, tools
- Access: repos, environments, groups
- Equipment: laptop, peripherals, security setup
- 30/60/90: goals and success criteria

## Owners
- DRI: People Ops; collaborators: IT, Hiring Manager

## Security & Privacy
- Do not store personal IDs here; track in HRIS.
